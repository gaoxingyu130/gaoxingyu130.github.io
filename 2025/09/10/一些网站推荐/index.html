一些网站推荐

GW2025-09-102025-09-10

🪄内容创作

  • 网站：https://onlineconvertfree.com/zh/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介：转化文件格式免费网站很多，这是我常用的一个，功能丰富，图片文档视频电子书所有的转换都集成了
  • 网站：https://www.watermarkremover.io/zh
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 免费去除图像中的水印，对于简单的文字水印的去除能力相当不错
  • 备注 : 需科学上网
  • 网站：https://www.erase.bg/zh
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 采集图像的背景使之透明，提取主体部分，效果非常不错
  • 备注 : 需科学上网
  • 网站：https://huemint.com/brand-1/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 对于颜色分布较简单的图像，可以自由编辑其组成颜色
  • 网站：https://docsmall.com/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介: 一个网页图像加载的速度很大程度上取决于图像原始大小，本网站的图像压缩服务非常强大，在几乎不影响视觉分辨的程度下能将原始图像压缩到很小的程度。 本博客上的大部分风景图片都是经过压缩的。
  • 网站：https://www.ldmap.net/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 地图标记，可以自定义点、线、面及地图样式等，相对于百度高德、谷歌地图等，它的标记功能更加丰富，自定义程度相当不错。
  • 缺点：免费版最多只有100个标签，且不支持进口导出，VIP收费过于昂贵（针对企业的），个人版“惠员”为99元每年，但也只支持300个标签。 我很长一段时间以来一直想用其他工具来代替它，但始终没有找到满意的。 所以，就着用吧~
  • 网站：https://app.diagrams.net/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 免费的绘图工具、开源制作思维导图和流程图等非常方便
  • 网站：https://www.qrcode-tiger.com/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 生成你想要的网址链接 ( URL ) 的二维码，二维码形式可以高度自定义

💎数学工具

  • 网站：https://www.geogebra.org/?lang=zh-CN
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 最好的几何绘图工具！
  • 网站：https://www.latexlive.com/home##
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : LaTeX 输入数学公式，这里提供了便捷的工具和公式模板，输入和预览响应速度很快，帮助文档也写得很好，各种语法点都写得很详细，以及可扩展插件。
  • 缺点：编辑和预览区域上下排布，只看这两个窗口的时候就没有工具栏了，还是感觉左右排布更方便
  • 网站：https://www.hostmath.com/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : LaTeX工具，提供公式模板，界面简单语音，滚动滚动翻页，响应速度快

🎣 资源下载

  • 网站：zlibrary（https://zh.z-library.se/，singlelogin.re)
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 下载图书，伟大消耗多言！
  • 缺点 : 由于美观的原因，各种 url 经常被封…
  • 网站：https://www.y2mate.com/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 下载 YouTube 视频，速度挺不错，免费方便
  • 缺点：暂时无明显槽点，就怕哪一天就G了
  • 网站：https://savetwitter.net/zh-cn
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 下载 Twitter 视频，速度挺不错，免费方便
  • 背面：同上
  • 网站：https://flac.life/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 可以免费下载音乐，优点是无广告使用方便，每首歌曲可以选择下载普通/极高/无损flac的音质。
  • 缺点 : 每 12 小时要在其公众号上发消息获取验证码才能自由下载
  • 网站：全国图书馆参考咨询联盟(superlib.net)
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 不能像Z-library那样自由下载图书，但是资源十分丰富，很多Z-library上没有的中夹这里齐全找到，可以申请每次至多50页的资源链接发送到你的邮箱里，响应速度很快，多试几次就可以…白嫖一书了
  • 网站：https://btbtt11.com/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 很著名的种子下载网站，很多电影都可以找到资源, 或网盘链接等
  • 网站：https://meta.appinn.net/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 发现和分享小众软件的论坛，情景的好用小软件及配置，也许可以方便你的生活呢

✨ 壁纸与绘画

  • 网站：https://wallspic.com/cn
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 大量极高质量壁纸，免费无广告登陆，强烈推荐！
  • 网站：https://pixabay.com/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 壁纸网站，免费的资源已经很丰富了，可以下载到非常清晰的图
  • 网站：https://wallhaven.cc/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 壁纸数量多，免费无广告免登陆，太棒了！ ( 电脑桌面 )
  • 网站：https://bz.zzzmh.cn/index
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 免费无广告, 壁纸 ( 电脑桌面壁纸 )
  • 网站：https://bizihu.com/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介: 质量非常非常高！ 网站设计优雅，免费无广告免登陆，强烈推荐(手机竖版壁纸)
  • 网站：https://gallerix.asia/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 拥有十分丰富的名画资源，免费下载
  • 网站：https://www.nbfox.com/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介: 网站前端设计很不错，名画资源十分丰富，可免费下载的资源很多

😀小游戏

  • 网站：https://poki.com/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 有非常多的经典小游戏，无广告，体验很不错，让我想起十年前的4399最好的时候~
  • 缺点：很多游戏无法全屏
  • 网站：https://itch.io/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 非常丰富的游戏资源库
  • 网站：https://www.jxgame.net/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : 二十多个经典小游戏~

🪐 开发者

  • 网站：https://www.v2ex.com/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介 : V2EX = 探索方式。 这是一个可以讨论技术问题、分享知识和观点的论坛，成员主要是互联网、游戏和媒体行业的从业者。 网页UI简洁，社区氛围浓厚，无广告。
  • 网站：https://www.helloworld.net/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介：域名很棒，这是一个较新的国内开发者社区，UI很直观，无广告，浏览不错的体验。 帖子多为技术分享，质量不错。 还有很多视频和文字教程。
  • 网站：https://segmentfault.com/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介: 思否. 面向开发者的问答社区。可以搜索和交流编程技术相关的问题。 UI设计相当，广告很少，阅读体验不错。 问答有周榜、月榜和极侦点。 很容易找到质量高的问答。
  • 网站：https://www.oschina.net/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介：国内最大的开源技术社区。 由开源软件库、代码分享、资讯、讨论区和博客等几大频道内容组成，综合质量不错。 （个人感觉UI设计有待优化，文字总堆在一块）
  • 网站：https://linux.do/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 学习、交流软件、生活、娱乐方面的知识论坛，体验轻松愉悦，还能发现最新最快的羊毛信息，感受有取代V2EX的趋势。
  • 网站：https://forum.ywhack.com/
  • 简介 : 网络安全行业必用

🌏涨知识

  • 网站：https://www.jetpunk.com/
  • 推荐指数 : ⭐⭐⭐⭐
  • 简介: 一个包含地理、历史、语言、艺术…杂七杂八知识的问答测试网站（全是测验！）偶尔无聊我就去填一填世界地图…中国地级市之类的，其实已经到了看不到地图填满的程度了（保命声明：本人对一些台湾省恶意分离开的行为将不予评价）
  • 网站：https://xiaoce.fun/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介: 也是百科知识填空问答网站，可以对照上面那个汉化版，也有很多独到的问题！ （其实也有同名微信小程序）

💠 刊物

  • 网站：https://nav.ops-coffee.cn/weekly.html
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介：集成了很多高质量周刊的入口。
  • 网站：https://zishu.me/
  • 推荐指数 : ⭐⭐⭐⭐⭐
  • 简介 : 作者经常混迹于 GitHub，LinuxDo，V2EX，等各大技术社交论坛，喜欢折腾，研究各种奇怪的程序和网站。同时将周刊一起发布到这个网站，获取更新的知识，无论是技术类型还是娱乐，让自己的钱源不断 + 1
  • 网站：https://www.ruanyifeng.com/blog/
  • 推荐指数 : ⭐⭐⭐⭐⭐
